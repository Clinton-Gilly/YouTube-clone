## Installation

1. Clone the repository
    ```
    git clone https://github.com/yourusername/youtube-clone.git
    ```
2. Navigate to the project directory
    ```
    cd youtube-clone
    ```
3. Install dependencies
    ```
    npm install
    ```
4. Start the server
    ```
    npm start
    ```
